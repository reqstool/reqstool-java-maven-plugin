== Usage

To use the Reqstool Java Maven Plugin, add the following configuration to your Maven project's `pom.xml`:

```xml
<build>
    <plugins>
        <plugin>
            <groupId>io.github.reqstool</groupId>
            <artifactId>reqstool-maven-plugin</artifactId>
            <version>0.1.0</version>
            <executions>
                <execution>
                    <goals>
                        <goal>assemble-and-attach-zip-artifact</goal>
                    </goals>
                </execution>
            </executions>
            <dependencies>
                <dependency>
                    <groupId>io.github.reqstool</groupId>
                    <artifactId>reqstool-java-annotations</artifactId>
                    <version>0.1.0</version>
                </dependency>
            </dependencies>
        </plugin>
    </plugins>
</build>
```

The default `<phase>` the plugin runs during is `verify`, this is done to ensure all test files are generated before the plugin runs.
